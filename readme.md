# Azure AI Foundry Computing Historian Agent

## Overview

This project demonstrates the development of an AI agent using Microsoft Azure AI Foundry. The agent specializes in computing history and can answer questions about historical computing systems, pioneers, and technological developments.

## Features

* Azure AI Foundry Agent
* OpenAI Responses API integration
* Azure Entra ID authentication
* Flask-based web interface
* Conversation history management
* Web search-enabled responses

## Technologies Used

* Python
* Flask
* Microsoft Azure AI Foundry
* Azure Identity
* OpenAI SDK
* Git & GitHub

## Project Architecture

User → Flask Web App → Agent Client → Azure AI Foundry Agent → Response
